# mh5-leyu-hub

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面资源的存储与展示空间。所有页面均以独立文件形式存在，便于浏览、分享和版本管理。

## 目录说明

```
mh5-leyu-hub/
├── pages/        # 页面文件存放目录
├── assets/       # 公共资源文件（样式、脚本、图片等）
├── index.html    # 入口页面
└── README.md     # 本文件
```

- `pages/`：存放所有独立 HTML 页面，每个页面为一个完整的功能单元。
- `assets/`：存放页面共享的静态资源，如 CSS、JavaScript、图像等。
- `index.html`：仓库的根入口文件，可用于导航或展示目录。

## 页面归档说明

本仓库中的 HTML 页面均以独立文件形式归档。每个页面包含完整的 HTML 结构，无需依赖外部服务器环境即可在浏览器中直接打开使用。页面之间无直接关联，各自独立运行。

归档页面可能包含但不限于：
- 信息展示页
- 工具类页面
- 学习笔记页
- 实验性页面

## 维护说明

- 本仓库定期更新，新增或优化已有页面。
- 页面文件命名应简洁明了，建议使用英文小写加连字符（如 `example-page.html`）。
- 资源文件存放于 `assets/` 目录，避免与页面文件混放。
- 欢迎提交 Issue 或 Pull Request 进行内容补充或问题反馈。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意页面。
3. 也可通过 GitHub Pages 等静态托管服务部署后在线访问。
